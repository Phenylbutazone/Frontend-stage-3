# Frontend-stage-3

## How to run the project

You can run this project on **any machine** that has a web browser.

### 1. Clone or download the repo

```bash
git clone https://github.com/Phenylbutazone/Frontend-stage-3.git
cd Frontend-stage-3
```

Or download the ZIP from GitHub and extract it, then open that folder.

### 2. Open in a browser (simplest)

1. Inside the project folder, locate `index.html`.
2. Double‑click `index.html` (or right‑click → **Open With…** → your browser).
3. Your browser will open the **Featured iPhone Accessories** page.

### 3. Run a simple local server (recommended)

From a terminal in the project folder:

```bash
python3 -m http.server 8000
```

Then open your browser and go to:

```text
http://localhost:8000
```

### 4. Using VS Code

1. Open the cloned folder in VS Code.
2. Install and enable a **Live Server** extension.
3. Right‑click `index.html` and choose **"Open with Live Server"**.
